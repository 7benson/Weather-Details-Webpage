

A Backend-web-development concentrated work with openweatherorg API to get the weather details of the city name 
you enter on the webpage.

To test the files, please install the required node modules mentioned in package.json file ,


Testing
After hosting it on the local server, head over to localhost:(port_number) , enter the city name ,
 and get the weather conditions of that place.

Please make sure you add your api key from openweatherorg , to the field specified in the app.js file
